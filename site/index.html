XDK
eXtensible markup language Design Kit
API Docs · Download · Rubyforge
XDK is essentailly a Ruby-based XML schema DSL. It provied a means of defining the structure of valid XML formats, such as RSS and ATOM feeds. With these definitions, one can then use them as document validators, fully accessable object representations, or as document builders using Ruby blockup notation.

XDK is still in alpha stage of development. While some of it works quite well (such as builders), other functionality is still under heavy contruction.

Blockup Example

Blockup notation is Blow's coined term for using Ruby blocks as in-code markup DSL. Here's a very simple example: 


        require 'blow/html'

        puts Html.document {
          body {
            h1 "Hello"
          }
        }
      
        <html>
          <body>
            <h1>Hello</h1>
          </body>
        </html>

Pretty simple, yes? This is very much like the Builder and Markaby projects. But Blow is more flexible. Plus, Blow has many other useful libraries.

Anonymous Subversion Access

This project's SVN repository can be checked out through anonymous access with either of the following command(s).

      svn checkout http://blow.rubyforge.org/svn/
      svn checkout svn://rubyforge.org/var/svn/blow

Developer Subversion Access via SSH

Only project developers can access the SVN tree via this method. SSH must be installed on your client machine. Substitute developername with the proper values. Enter your site password when prompted.

      svn checkout svn+ssh://developername@rubyforge.org/var/svn/blow
